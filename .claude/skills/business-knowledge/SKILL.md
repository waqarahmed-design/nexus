## Core principle

Design decisions must be justified in both user value AND business value. Ask:
- What user behavior does this drive?
- How does that behavior map to a business metric?
- What is the expected impact on revenue, retention, or acquisition?

## Quick reference: key frameworks

- **Business Model Canvas** — value prop, channels, revenue streams, cost structure
- **Unit Economics** — CAC < LTV (target 3:1 ratio), payback period
- **Pirate Metrics (AARRR)** — Acquisition, Activation, Retention, Revenue, Referral
- **North Star Metric** — the single metric that best captures the value you deliver
- **RICE Scoring** — Reach x Impact x Confidence / Effort for prioritization
